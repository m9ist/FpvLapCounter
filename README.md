# FPV Lap Counter

Streamlit-приложение для анализа FPV-видеозаписей: автоматически находит пролёты через ворота, считает времена кругов и ищет лучшие серии подряд.

![Python](https://img.shields.io/badge/python-3.10%2B-blue)
![Streamlit](https://img.shields.io/badge/streamlit-1.40%2B-red)

## Возможности

- 📁 Пакетный анализ папки с видео (`.ts`, `.mp4`, `.avi`, `.mov`, `.mkv`)
- 🧠 Детекция ворот через эмбеддинги (CLIP / MobileNet / EfficientNet)
- 🖼 Референсные кадры ворот: загрузка из файла, буфера обмена, истории
- 📈 Интерактивный график сходства с кликом по пику → переход к кадру
- ✅ Верификация кандидатов: подтвердить / отметить как фейк / пакетные действия (все ✅, все ❌, сброс)
- 🏆 Таблица кругов, лучший одиночный круг и лучшие N подряд
- 📊 Сравнение нескольких видео
- 💾 Результаты сохраняются в JSON рядом с видео (`.fpv.json`)

## Установка

```bash
pip install -r requirements.txt
```

> CUDA-видеокарта используется автоматически для EasyOCR и torch-моделей.

## Запуск

```bash
streamlit run app.py
```

## Использование

1. В боковой панели выбери папку с видео
2. Загрузи референсные кадры ворот (JPG/PNG или из буфера обмена)
3. Настрой параметры детекции (порог сходства, минимальная длина круга и т.д.)
4. Нажми **🚀 Обработать** — начнётся анализ выбранных видео
5. Перейди на вкладку **✅ Верификация** и подтверди/отклони кандидатов
6. Смотри результаты на вкладке **🏆 Круги** и **📊 Сравнение**

## Архитектура

```
app.py                      — главный файл Streamlit
ui/
  sidebar.py                — боковая панель (настройки, референсы)
  video_list.py             — список видео с выбором и статусом
  graph_tab.py              — график сходства + просмотр кадров
  verify_tab.py             — верификация кандидатов-пролётов
  laps_tab.py               — таблица кругов и сравнение
core/
  detector.py               — GateDetector: эмбеддинги + поиск пиков
  models.py                 — реестр поддерживаемых моделей
  osd_reader.py             — OCR таймера из OSD (EasyOCR)
  lap_analyzer.py           — расчёт кругов, лучшие N подряд
storage/
  project.py                — загрузка/сохранение .fpv.json
  references.py             — RefImage: история референсных кадров
utils/
  logger.py                 — логирование в файл app.log
dev/
  verify_preview.html       — CSS-плейграунд для карточек верификации
```

## Как работает детекция

1. Для каждого N-го кадра вычисляется косинусное сходство эмбеддинга кадра с эмбеддингами референсных изображений ворот
2. Временной ряд сходства сглаживается (Savitzky-Golay)
3. Пики выше порога = кандидаты на пролёт через ворота
4. Для каждого кандидата OCR считывает время из OSD-таймера
5. Пользователь верифицирует кандидатов, после чего пересчитываются круги
